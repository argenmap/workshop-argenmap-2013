h2. 1.1. Crear un mapa

En argenmap, un mapa es una colección de capas, controles y marcadores para la interaccóin con el usuario. Un mapa es generado con tres ingredientes básicos: código HTML, declaraciones de estilo CSS y código de inicialización (JAVASCRIPT).

h3(#mapaejemplo). Ejemplo funcional

pre.. <!doctype html>
<html lang="es">
  <head>                              
    <script type="text/javascript" src="http://maps.google.com/maps/api/js?sensor=false"></script>
    <script type="text/javascript" src="http://code.jquery.com/jquery-1.8.2.js"></script>
    <script type="text/javascript" src="http://www.ign.gob.ar/argenmap/argenmap.jquery.min.js"></script>
    <style type="text/css">
      div#mapa {
        height:315px;
      }
    </style>
    <script type="text/javascript">
      $(document).ready(function(){
        $("#mapa").argenmap().centro(-32,-65).zoom(6);
      });
    </script>
  </head>
  <body>                              
    <div id="mapa"></div>                             
  </body>
</html>
