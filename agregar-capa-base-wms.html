4.3 Importar una capa base WMS

4.3.1 Configurar y pedir una capa base WMS

Importar una capa base WMS es muy similar a agregar una capa WMS superpuesta, las diferencias son principalmente funcionales.

Para importar una capa usás el método .agregarCapaBaseWMS(). Una de las grandes diferencias entre .agregarCapaWMS() y .agregarCapaBaseWMS() es que al agregar una capa base la misma sólo puede visualizarse cambiando la vista desde el menú desplegable de capas. Asimismo, la nueva capa base no tendrá ninguna otra capa superpuesta, por lo que, si querés visualizar otros datos, tendrás que agregarlos con más llamadas .agregarCapaWMS()

.agregarCapaBaseWMS() recibe como parámetro un objeto simple de Javascript con las siguientes opciones:

  • url: es la url donde el servidor WMS recibe los pedidos. En el XML del pedido GetCapabilities podés verlo buscando el nodo Capability.Request.GetMap.DCPType.HTTP.Get.OnlineResource:href
  • capas: un string separado por comas con los nombres de las capas que querés pedir al servidor. Las capas están listadas en el XML en Capability.Layer.Layer, y de ahí tenés que usar la propiedad Name
  • nombre: es el nombre que se le dará a la capa base. Este nombre aparecerá en el menú desplegable arriba a la derecha del mapa. También te servirá para hacer referencia a la capa en caso que necesites quitarla.

4.3.2 Ejercicio

  • Consultá el servidor WMS de formosa
  • Hacé click en Javascript y pegá el siguiente código (recordá pegarlo después de inicializar el mapa)
$('#mapa').agregarCapaBaseWMS({
  url:'',
  capas:'',
  nombre:''
});
  • Configurá la capa que estás agregando con los datos que obtengas del XML del WMS de Formosa (url del servidor, nombre/s de capa/s a pedir)
  • Mirá el resultado en el panel de la derecha

For your pleasure: ya ubicamos el mapa con centro en Formosa para que puedas visualizar la capa cuando la incorpores.
