1. Conceptos básicos de argenmap.jquery

argenmap.jquery es una librería javascript que sirve para construir aplicaciones con mapas en un browser.

Este taller cubre los detalles de argenmap.jquery v1 y da un pantallazo de la siguiente versión, argenmap.jquery v2 basada en la librería OpenLayers.

La librería permite integrar datos desde varias fuentes, ofrece una API amigable para usuarios familiarizados con jQuery y genera mapas compatibles con una alta gama de dispositivos (PC, mobile, tablets, etc).

Este módulo presenta los conceptos de argenmap.jquery fundamentales para crear un mapa.

En este módulo vas a crear un mapa, entender las partes del mapa y obtener links adicionales para mayor documentación.
